# 99_archive — superseded, not deleted

When a document is replaced or retired, move it here instead of deleting it. The audit trail of how our thinking evolved is worth more than the tidiness of an empty folder.

## How to archive

1. Move the file from its original location into a subfolder here that mirrors the original path (e.g., `99_archive/03_design/wireframes/old-builder-canvas.md`).
2. Add a short header at the top of the archived file:

```
> **Archived YYYY-MM-DD.** Superseded by [new file](../path/to/replacement.md). Reason: …
```

3. Update any incoming links in living documents to point to the replacement.

## What does *not* get archived

- Temporary working files. Delete those.
- Generated artifacts that can be regenerated from source. Regenerate them.
- Truly mistaken drafts that never informed a decision. Delete them; the noise is worse than the audit value.
